Sobre Textos

Textos são conjuntos de caracteres capazes de serem lidos. Interpretados? Nem sempre. Mas não é para discutir a respeito das propriedades metafísicas de um texto que este site foi feito.

O objetivo é mostrar que há várias tags em HTML que alteram um determinado texto. Algumas fazem coisas visualmente parecidas, mas possuem valores semânticos diferentes.

Exemplos:

  • B e Strong.
  • Input, com type button e Button.

Sério?

Sim!

Pera, os títulos estão ficando menores?

Sim, pois são subtítulos. Existem formas corretas de usar eles para o sentido semântico da coisa, e lembre-se: NÃO USE UM H1 SÓ PELO TAMANHO!
